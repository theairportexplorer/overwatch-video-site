Overwatch Database Form

Video URL (Required)

Website (YouTube/Vimeo) URL of video

YouTube/Vimeo iFrame (Required)

Generated embed <iframe /> code from YouTube/Vimeo to insert YouTube/Vimeo video players. Highly recommended Vimeo is used, because it doesn't throttle playback quality like YouTube.

Video Date (Required)

Date video was created. Must be in ISO 8601 (YYYY-MM-DD) format.

Hero Name (Required)

Must select hero name from drop list.

Alternate Title (Optional)

Alternate title for video to display. By default it will be:

<hero name>_<video date>
Clip Type (Optional)

Choose between "highlight" or "potg". By default it will be "highlight"

Tags (Optional)

Tag videos for cross referencing. Every tag must be space separated and start with a "#" sign.

Description (Optional)

Personal commentary, if desired.
